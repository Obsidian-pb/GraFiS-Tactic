<h1>Вставка значений из фигур:</h1>
<p>Для вствки значений из ячеек формул фигуры следует указать название ячейки охватив ее знаком $ </p>
<p><b>Например:</b><br/>
'$Площадь $' (<i>без пробела в конце</i>) - в итоговом тексте вместо этого выражения будет вставлено значение ячейки 'Prop.Площадь': <b>$Площадь$</b>.
</p>
<p style='color: red'>Важно - вставляются только значения ячеек из секции Prop (Данные фигуры)</p>

<hr/>
<h1>Пример разметки для формул:</h1>

<p align='center'><b>S<sub>ф</sub>=a x b</b></p>
<p>где S<sub>ф</sub> - площадь фигуры, мм<sup>2</sup>, a - длина фигуры, мм, b - ширина фигуры, мм</p>
<p align='center'>S<sub>ф</sub>=$Height$ x $Width$ = $Square$ мм<sup>2</sup></p>
